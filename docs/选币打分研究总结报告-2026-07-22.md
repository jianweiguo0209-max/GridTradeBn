# 选币因子可量化打分研究 · 总结报告

**日期**:2026-07-21 ~ 07-22 · **状态**:研究线收官 · **数据资产**:`data/score_research_2026-07-21/`
**结论一句话**:打分器本身判死(线性因子层三路全试尽),但研究衍生出已上线的 s030 引擎配置、
一套秒级验证管线、和三条有条件可用的信号残值。

---

## 1. 研究问题与方法(三道门框架)

**问题**:剥离建网/执行/止损,能否给币直接打"未来 12h 网格适配度"分,替代现役三因子排名?

**标签工厂**(机器无关的路径物理量,PIT 后视 12h,全票池×小时级,四窗 272 万行):
- `cross1` 燃料 = 1% 对数阶梯穿越数(回转次数代理)
- `drift` 毒药 = |净漂移|
- `mae` 逆行 = 最大单边位移
- `eff` 形状 = cross1/(1+100·mae)(单位逆行换到的回转)

**三道门流程**(便宜的门先过,赢家才烧引擎算力):
标签门(解析标签 IC)→ 拟合门(时间正向训练/测试,留出零接触)→ 引擎门(真引擎 pnl 直面)。

## 2. 核心实证发现

### 2.1 网格 pnl 的物理公式(全研究最稳健结构)
四窗引擎标定(2,800 真引擎格):**pnl ≈ 形状(eff) − 漂移(drift)**
(eff 秩相关 +0.31/+0.30/+0.36/+0.37,drift −0.27~−0.35,裸燃料≈0)。
即"跌了会弹 vs 跌了还跌"的标签化。⚠这是当期标签↔当期 pnl 的事后记账,定义了目标但不保证可预测。

### 2.2 现役排名的因子画像(四窗一致)
| 因子 | 角色 | 判 |
|---|---|---|
| Er_2 | **低波选择器,主宰组合**(IC 幅度 2.5-4×) | 防御有效,形状负贡献 |
| Sgcz_5 | 栈内唯一正形状(+0.06~0.07) | 双刃(毒药同+) |
| Reg_v2_5 | 温和防御 | 中性 |
| S_shape_5 | 形状负(事件币形状差) | 与门控战役自洽 |
| Atr_5(未入栈) | **最强形状(+0.16~0.22)+最强燃料(+0.58~0.68)预测器** | 但燃料-毒药捆绑(drift +0.30~0.40) |

现役 rank_sum:防御向成立(避漂移/逆行),**但对形状与燃料是盲的**(eff IC ≈0~−0.06);
另有尾部病理:线性 IC 防御向但 top-12 实选桶毒药 +8~17%(单因子极端者当选,无共识门)。

### 2.3 打分器判死的完整证据链
1. **Phase2 线性拟合**:标签层大胜(y-IC 达现役 8×)但引擎门全灭(score↔pnl ≈0,训练窗都负)。
   机制:可预测成分全在"安静度"方向,静币无成交≈零 pnl——**可预测部分与可获利部分正交**。
2. **消融反证**:换纯网格引擎(E0)打分器更死(score↔pnl_E0 W1 −0.155)——不是引擎冤枉了信号。
3. **13 特征终诊断**:对引擎 pnl 无一过 |IC|>0.08 四窗同号线(现役引擎与纯引擎口径皆然)。
4. **统一解释**:线性特征粒度下截面 pnl 面是平的;现役排名的边缘活在极端尾部×止损链交互里,
   不在光滑梯度里——这也是七轮参数战役"现值即甜点"的理论收口。

### 2.4 几何配合下的"近突破"与留出否决
密格几何(c16/c26)使 pnl 变成燃料驱动(低漂移半样 pnl↔cross1 达 +0.55~0.74)→
**p12_cross1(过去 12h 已实现燃料)对宽带密格 pnl 四窗过线**(+0.215/+0.094/+0.093/+0.134,
全研究首个)→ 但留出门 FAIL(HOLD-B +0.067<+0.08)。终判:**真方向、弱幅度**——
六窗全正号(零假设下 p≈1.6%,非幻影),幅度不足以单独立项。

## 3. 可利用价值的成果盘点

### 3.1 已兑现(最大变现)
打分研究的机制发现(引擎抹掉燃料溢价/pv 误杀准赢家/宽带=便宜尾部保险)经组合级战役
转化为 **s030 配置,已上线 mainnet(2026-07-22)**:六窗对旧现值 Σ+7.1pp、双留出全过、
总量+稳健双料冠军。研究线名义上"零改动收官",实际产出了本项目迄今最大的一次配置升级。

### 3.2 可直接复用的基础设施(研究提效资产)
| 资产 | 用途 | 成本 |
|---|---|---|
| `phase2_score.py` 三层门管线 | **今后任何选币想法先过标签门/引擎门再谈实盘**,防"标签好看引擎死"的老坑 | 秒~分钟级 |
| 272 万 PIT 标签 + 全池因子面板(sc_*) | 新信号的即时 IC 检验床 | 已落盘 |
| 2,800 引擎标定格(ec_*)+消融五档(ab_*) | 引擎口径的 pnl 参照系 | 已落盘 |
| `holdout_gate.py` | 留出验证管线(标签/因子/引擎全链重建,HOLD-A/B 零接触纪律) | ~50min |
| `market_r15.parquet` + `build_market_r15.py` | 636 币截面中位 15m 收益指数(日历对齐),任何截面信号的市场基准 | 已落盘 |
| 引擎 pv_idio 条件化钩子(默认关) | 未来任何"条件化止损"想法即插即测 | 零成本待命 |

### 3.3 有条件可用的信号残值(需新样本再验,勿直接部署)
1. **p12_cross1(燃料持续性)**:六窗全正、幅度 0.07~0.22。不够做主排名,但可作
   **负向过滤候选**(剔除燃料枯竭币,即 top-12 内的"未触网预备队"——几何诊断显示静币
   触不到网是真实亏损源)。启用前置:攒 3 个月新数据做一次干净的 out-of-time 验证。
2. **Atr_5 的双面性**:最强燃料/形状预测器但捆绑毒药——若未来做非线性模型,它是首选
   特征;线性权重里勿单独加(会连毒药一起买)。
3. **崩盘两态论**(同步 V 型持有赢 / 离散单雷早割赢):判别器组合级不划算已证,但该
   知识对**手动风控**有值:大盘同步暴跌时不宜手动加码止损,单币独立暴跌时反之。

### 3.4 明确不可用(别再试,已付学费)
- 线性因子打分(三路全试尽:直拟/几何配合/留出)
- 量能尖峰幅度做单币/市场判别(已证反向:大冲洗日全币同飙)
- 固定阈值/波动归一的 idio 判别器全参数族(六窗完备否决,IS 误杀恒吞收益)
- s 因子条件化族(前役已否)

### 3.5 复活条件(如果将来重启打分线)
非线性/尾部方法(树模型抓极端尾部×止损链交互)或新信息源(订单簿深度、资金流、清算数据);
且必须走三道门流程+双留出,判据预注册。

## 4. 方法论沉淀(比结论更值钱的部分)
1. **引擎门先行**:标签/IC 层的胜利与引擎 pnl 可以完全正交——任何选币研究先花几秒过引擎门。
2. **留出窗只验收不选参**:pvloose(判定窗 Σ+9.5pp)与 idioZ3(三窗全场最优)都死在留出——
   本研究期两次重演 sx0.003 剧本,纪律再次自证。
3. **锚逐位复现守卫**:每窗先跑锚并 byte 级比对存档,引擎改动零污染有实证(pv_idio 全链回归)。
4. **预注册判据**:判线(|IC|>0.08 四窗同号、两留出均 >+0.08)在跑前写死,赛后不挪门柱。
5. **配对消融**(同样本逐层回加)是归因的黄金标准:止损链 71% 期望税、回撤止盈没收燃料
   溢价等结论均由此获得,观测性分析给不出。
